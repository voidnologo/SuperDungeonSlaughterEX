# Proposals & Planning Documents

This directory contains design documents, implementation plans, and feature proposals for Super Dungeon Slaughter EX.

## Documents

### Implementation & Architecture
- **[IMPLEMENTATION_PLAN.md](./IMPLEMENTATION_PLAN.md)** - Original implementation plan for porting from Python to Elixir Phoenix LiveView
  - Core architecture decisions
  - Data models and game flow
  - UI/UX design
  - Phase-by-phase implementation guide

### Feature Proposals
- **[feature_expansion_ideas.md](./feature_expansion_ideas.md)** - Comprehensive list of proposed features for game expansion
  - 12 proposed features across different categories
  - Implementation notes and priorities
  - Features #5 (Boss Encounters) and #10 (Stat Dashboard) selected for initial implementation

- **[monster_scaling_proposals.md](./monster_scaling_proposals.md)** - Detailed analysis of monster scaling approaches
  - Three options: Expanded list, dynamic scaling, or hybrid
  - Balancing formulas and calculations
  - Comparison matrix and recommendations

## Current Focus

**Selected for Implementation:**
1. **Boss Encounters & Floor Transitions** - Add structured progression with boss fights every 5-10 levels
2. **Enhanced Stat Tracking Dashboard** - Expand statistics system with lifetime tracking and analytics

## Usage

These documents serve as:
- Reference for implementation decisions
- Design specifications for new features
- Historical record of architectural choices
- Planning resources for future development
